Fungsi dari code ini adalah untuk menghitung total biaya pengiriman paket di sebuah ekspedisi. 
Cara kerja program ini adalah user menginput berat paket, jarak paket dari asal ke tempat tujuan serta menginput panjang, lebar dan tinggi paket, setelah itu komputer akan menjalankan program dengan 1. menghitung volume terlebih dahulu; 2. Lalu menghitung biaya dasar; 3. Menghitung apabila ada biaya tambahan; 4.Menampilkan output berupa harga dari pengiriman paket tersebut.
